# Step 5: 사용자 선택

## 🎯 이 단계의 목표
사용자에게 TOP 5 도구를 제시하고, **선택을 대기**합니다.

---

## 📋 사용자에게 제시할 형식

```
╔═══════════════════════════════════════════════════════════════╗
║  🎯 추천 개발 도구 TOP 5 (100개 중 자동 선별)                    ║
╚═══════════════════════════════════════════════════════════════╝

1️⃣ [8.10점] SVG to JSX 변환기
   📂 카테고리: converter
   🔧 난이도: medium
   💡 이유: 디자인 시스템 구축 시 SVG를 JSX로 변환하는 작업이 빈번함
   🎯 롱테일 키워드: "React SVG JSX 변환 자동 최적화 도구"

2️⃣ [7.50점] API Mock 데이터 생성기
   📂 카테고리: generator
   🔧 난이도: high
   💡 이유: 프론트엔드 개발 시 백엔드 API 대기 중 Mock 데이터 필요
   🎯 롱테일 키워드: "TypeScript 타입 기반 API Mock 자동 생성"

3️⃣ [7.20점] SQL 쿼리 포맷터
   📂 카테고리: formatter
   🔧 난이도: medium
   💡 이유: 복잡한 SQL 쿼리를 읽기 쉽게 정리
   🎯 롱테일 키워드: "SQL 가독성 향상 문법 검증 자동 포맷팅"

4️⃣ [7.00점] 번들 크기 분석기
   📂 카테고리: utility
   🔧 난이도: high
   💡 이유: 프로젝트 빌드 시 번들 크기 확인 및 최적화 제안
   🎯 롱테일 키워드: "Next.js React 번들 크기 최적화 가이드"

5️⃣ [6.80점] Postman to Fetch API 변환기
   📂 카테고리: converter
   🔧 난이도: high
   💡 이유: Postman Collection을 실제 코드로 변환할 때 필요
   🎯 롱테일 키워드: "Postman API 테스트 Fetch 코드 자동 생성"

───────────────────────────────────────────────────────────────
💬 선택 옵션:
  • 1-5 입력: 해당 도구 생성 시작
  • 'more': 다음 5개 도구 보기 (6-10위)
  • 'all': 전체 100개 목록 보기
  • 'exit': 종료
───────────────────────────────────────────────────────────────
```

---

## 🎮 사용자 입력 처리

### 입력 1: 숫자 1-5
→ 선택한 도구 정보를 `automation/cache/selected-tool.json`에 저장
→ **Step 6-1: tool.config.ts 생성**으로 이동

### 입력 2: 'more'
→ 6-10위 도구 표시
→ 다시 선택 대기

### 입력 3: 'all'
→ 전체 100개 목록을 카테고리별로 표시
→ slug 직접 입력 또는 순위 선택

### 입력 4: 'exit'
→ 프로세스 종료

---
