# 팀 프로젝트 발표 자료: Codepis (코드피스)

## 1. 프로젝트 개요 (Overview)

*   **프로젝트명:** Codepis (코드피스)
*   **슬로건:** 개발자를 위한 하이엔드 온라인 도구 모음 (High-End Developer Tools)
*   **프로젝트 목적:** 개발자들이 실무에서 반복적으로 마주치는 문제들을 해결하고, 생산성을 극대화할 수 있는 고품질의 무료 온라인 도구를 제공하는 것.
*   **핵심 가치:**
    *   **생산성 (Productivity):** 반복 작업 자동화 및 시간 단축
    *   **품질 (Quality):** Google이 인정하는 고품질 콘텐츠 및 도구 제공 (SEO 최적화)
    *   **접근성 (Accessibility):** 설치 없이 웹 브라우저에서 즉시 사용 가능

## 2. 문제 정의 및 해결 방안 (Problem & Solution)

### 🚨 문제점 (Pain Points)
1.  **반복적인 수작업:** JSON 데이터를 TypeScript 인터페이스로 변환하거나, Tailwind 클래스를 찾기 위해 매번 검색하는 등 단순 반복 작업에 시간이 소요됨.
2.  **도구의 파편화:** 필요한 도구들이 여러 웹사이트에 흩어져 있어 관리하기 어렵고, 광고가 많거나 UI가 불편한 경우가 많음.
3.  **데이터 보안 우려:** 외부 도구 사용 시 민감한 코드나 데이터 유출에 대한 우려가 존재함.

### ✅ 해결 방안 (Our Solution)
*   **All-in-One 플랫폼:** 개발자에게 필요한 핵심 도구들을 하나의 플랫폼에서 제공.
*   **직관적인 UI/UX:** ShadCN UI와 Tailwind CSS를 활용하여 깔끔하고 현대적인 사용자 경험 제공.
*   **Client-Side 처리:** 대부분의 데이터 처리를 브라우저 내부에서 수행하여 서버로 데이터를 전송하지 않아 보안성 강화.
*   **고품질 가이드 제공:** 단순 도구 제공을 넘어, 각 도구의 활용법과 기술적 배경을 설명하는 SEO 최적화된 콘텐츠 제공.

## 3. 주요 도구 소개 (Key Features)

### 🛠️ 1. JSON to TypeScript 변환기 (JSON to TS)
*   **기능:** JSON 데이터를 입력하면 즉시 TypeScript 인터페이스(Interface)와 타입(Type) 정의로 변환.
*   **차별점:**
    *   **Zod 스키마 자동 생성:** 런타임 데이터 검증을 위한 Zod 스키마까지 함께 생성하여 안전한 데이터 처리 지원.
    *   **중첩 객체 자동 분리:** 복잡한 중첩 구조도 자동으로 분리하여 가독성 높은 타입 정의 생성.
    *   **실시간 변환:** 입력과 동시에 결과 확인 가능.
*   **사용자 이점:** 백엔드 API 연동 시 타입 정의 시간을 획기적으로 단축하고, 휴먼 에러 방지.

### 🎨 2. Tailwind 클래스 시각화기 (Tailwind Class Visualizer)
*   **기능:** Tailwind CSS 클래스를 입력하면 해당 클래스가 실제로 어떤 스타일(CSS 속성)을 적용하는지 실시간으로 분석하고 시각화.
*   **차별점:**
    *   **실시간 미리보기:** 코드 입력 즉시 렌더링 결과 확인.
    *   **상세 해설:** 각 클래스의 의미와 역할을 상세하게 설명하여 학습 효과 제공.
    *   **반응형 접두사 지원:** `md:`, `lg:` 등 반응형 클래스도 정확하게 해석.
*   **사용자 이점:** Tailwind CSS 초보자의 학습 도구로 활용 가능하며, 복잡한 클래스 조합을 디버깅할 때 유용.

### 📉 3. 토큰 슬리머 (Token Slimmer)
*   **기능:** AI 모델(LLM)에 입력할 프롬프트의 토큰 수를 줄여주는 최적화 도구.
*   **차별점:**
    *   **스마트 압축:** 의미를 유지하면서 불필요한 공백, 조사 등을 제거하여 토큰 사용량 절감.
    *   **토큰 카운터:** 압축 전후의 토큰 수와 절약 비용을 실시간으로 계산.
*   **사용자 이점:** API 비용 절감 및 제한된 컨텍스트 윈도우(Context Window) 효율적 활용.

### 📸 4. 코드 스냅샷 (Code Snapshot)
*   **기능:** 작성한 코드를 예쁜 이미지로 변환하여 소셜 미디어(SNS)나 블로그에 공유하기 좋게 만들어줌.
*   **차별점:** 다양한 테마와 문법 강조(Syntax Highlighting) 지원, 고해상도 내보내기.
*   **사용자 이점:** 기술 블로그 작성이나 코드 공유 시 시각적 퀄리티 향상.

## 4. 기술 스택 (Tech Stack)

*   **Frontend Framework:** Next.js 16 (App Router)
*   **Language:** TypeScript
*   **Styling:** Tailwind CSS v4, ShadCN UI
*   **State Management:** Zustand
*   **Testing:** Vitest
*   **Deployment:** Vercel

## 5. 기대 효과 (Expected Impact)

1.  **개발 생산성 향상:** 단순 반복 작업을 자동화하여 개발자가 핵심 로직 구현에 집중할 수 있는 환경 조성.
2.  **코드 품질 개선:** 타입 안정성 확보(JSON to TS) 및 스타일 가이드 준수(Tailwind Visualizer)를 통해 전반적인 코드 품질 향상.
3.  **지식 공유 활성화:** 코드 스냅샷 등을 통해 개발 지식을 더 쉽고 예쁘게 공유하는 문화 확산.

## 6. 향후 계획 (Future Roadmap)

*   **도구 지속 추가:** 정규식 생성기, SQL 포맷터 등 개발자 필수 도구 지속 업데이트.
*   **사용자 계정 시스템:** 설정 저장 및 도구 사용 기록 관리 기능 도입 (Supabase 활용).
*   **커뮤니티 기능:** 도구 사용 팁 공유 및 피드백 수렴을 위한 게시판 기능 추가.
